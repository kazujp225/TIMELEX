## アニメーション仕様

### トランジション

```css
/* ホバー/アクティブ状態 */
transition: all 200ms cubic-bezier(0.4, 0, 0.2, 1);

/* サイドバー幅変更 */
transition: width 300ms cubic-bezier(0.4, 0, 0.2, 1);

/* ツールチップ表示 */
transition: opacity 200ms, visibility 200ms;
```

### イージング

```
標準: cubic-bezier(0.4, 0, 0.2, 1)
高速: cubic-bezier(0.4, 0, 1, 1)
```

## 参考スクリーンショット

### スタッフポータル - デスクトップ（展開）
- ロゴ: TUMELEX（グラデーション）
- ロールバッジ: 「スタッフ」（青）
- メニュー: ダッシュボード、予約一覧、カレンダー
- フッター: メールアドレス、ログアウトボタン

### 管理者ポータル - デスクトップ（展開）
- ロゴ: TUMELEX（グラデーション）
- ロールバッジ: 「管理者」（青）
- メニュー: ダッシュボード、スタッフ管理、相談種別、お問い合わせ元、アンケート、レポート、設定
- フッター: メールアドレス、ログアウトボタン

### モバイル - ボトムナビゲーション
- 固定下部バー
- アイコン + ラベル（短縮版）
- 最大4項目表示
- アクティブ項目には下部に青いインジケーター
